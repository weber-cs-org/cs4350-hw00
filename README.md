### 01 - Conditionals - Points **195**

#### The `if` Statement

Write a PHP function `Tgif($day)` that will print out `"Have a nice weekend!"` if **`$day`** is `"Fri"`.

#### The Ternary Statement

#### The `if...elseif...else` Statement
